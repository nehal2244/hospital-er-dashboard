# 🏥 Hospital ER Analytics Dashboard

This project is all about turning raw hospital data into meaningful insights.

Using 9,216 Emergency Room patient records, I built an interactive dashboard to understand what’s really happening inside an ER — from patient flow and wait times to critical cases and department performance.

---

## 🚀 Project Overview

Hospitals generate a huge amount of data every day, but without proper analysis, it becomes difficult to make quick and effective decisions.

In this project, I transformed raw data into a clean, interactive dashboard that highlights:
- Key patterns  
- Bottlenecks  
- Performance gaps  

What makes this project even more interesting is that the entire workflow — from data cleaning to dashboard creation — was supported using Generative AI, making the process faster and more efficient.

---

## 📊 Key Features

- Clear KPI cards (Total Patients, Avg Wait Time, Admission Rate)
- Interactive charts for trend analysis
- User-friendly filters (slicers):
  - Department  
  - Patient Severity  
  - Gender  
  - Age Group  
  - Admission Status  
  - Date Range  
- Heatmap to identify peak hours and busiest days  
- Visual breakdown of wait times and case severity  

---

## 📈 Key Insights

- ~50% of patients were admitted → Indicates high hospital load  
- Neurology had the highest wait time (36.8 mins) → Possible bottleneck  
- ~58% of patients were walk-ins → Suggests gaps in primary care access  

---

## 💡 Business Impact

This dashboard helps answer critical questions like:
- When does the hospital need more staff?  
- Which departments require attention?  
- How can patient wait times be reduced?  

Overall, it provides clear direction for improving efficiency and patient experience.
